# Basic-Calculator
This project is a Basic Calculator developed using the C programming language. It performs fundamental arithmetic operations such as addition, subtraction, multiplication, and division through a simple menu-driven interface.

A simple menu-driven basic calculator program written in C language. This project performs basic arithmetic operations and is suitable for beginners learning C programming.

# Description

The Basic Calculator allows users to perform fundamental mathematical operations such as addition, subtraction, multiplication, and division.
The program uses a switch-case structure and runs in a loop until the user chooses to exit.

This project helps beginners understand:

User input/output

Conditional statements

Looping

Functions (optional, if used)

## Features

Menu-driven interface

Supports basic arithmetic operations:

**Addition**

**Subtraction**

**Multiplication**

**Division**

Repeats until user exits

Easy-to-understand and beginner-friendly code

### Technologies Used

**Language:** C
